Procedure to Create and View a Website on the Production Server

  1. Open up Terminal
  2. Make sure your in your repository folder, and enter in the command "python -m SimpleHTTPServer" This will start the development server.
  3. Create your html folder "gedit 'filename.html'
  4. Now go back to google and put in "localhost:8000/filename.html" You won't be able to see any changes unless you have done some editing to the html folder.
  5. Start editing your folder, and remember the html format. Open up the template.html file if you need to. Copy and paste the template into your file, and add in your information. Remember that everytime you make a change to your html folder, you save it.
  6. Refresh your page every so often, and your changes should appear.
  7. To style your html page, you need to open up a css file. To do this you must put in the command: "gedit 'filenameofhtml.css'
  8. Go back to your html folder and under 'meta charset' put in the link 'link rel="stylesheet" href="filename.css" type="text/css"
  9. Now you can start editing your html on the css folder. Remember to style a certain part of your html you must put in the name of the section and the bracket "{" and to close it put in the bracket "}"
  10. Once you like what you have done you must save both your html and css to your repository folder with the commands "git add, git commit, and git push"
  11. To view it on the production server you put in the command: "sftp yournameandnumber@cmis-hsteacher" It will then as you for your password.
  12. It will ask you if you want to connect, type yes, and it will then say you have been connected to the production server.
  13. Make sure your in the public folder so put in the command: "cd and type p then hit tab" This will make put you into the public folder.
  14. Now type in the command: put 'filename" and your done.
